# Description

The `dcos marathon group list` command allows you to view the list of groups.

# Usage

```bash
dcos marathon group list [--json]
```

# Options

| Name |  Description |
|---------|-------------|
| `--json`   |  Displays JSON-formatted data. |.
| `-h`, `--help` | Display info about usage of this command. |
