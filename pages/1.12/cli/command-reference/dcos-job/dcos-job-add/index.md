# Description
Add a job.

# Usage

```bash
dcos job add <job-file> [OPTION]
```

# Options

None.

# Positional arguments

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `<job-file>`   |             |  A JSON-formatted job definition. |

# Parent command

| Command | Description |
|---------|-------------|
