# 说明
`dcos license renew` 命令允许您管理群集许可证。

# 使用

```bash
dcos license renew [OPTIONS] PATH
```

# 位置自变量

| 名称，简写 | 说明 |
|---------|-------------|
| | `PATH` | 包含许可证的文件的路径。|
