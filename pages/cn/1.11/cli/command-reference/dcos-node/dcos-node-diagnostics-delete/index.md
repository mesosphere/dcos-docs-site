# 说明
`dcos node diagnostics delete` 命令让您删除诊断捆绑包。

# 使用

```bash
dcos node diagnostics delete <bundle> [OPTION]
```

# 选项

无。

# 位置自变量

| 名称，简写 | 说明 |
|---------|-------------|
| `<bundle>`   |   The bundle filename. For example, `bundle-2017-02-01T00:33:48-110930856.zip`. |
