# 说明
`dcos marathon group list` 命令让您查看组列表。

# 使用

```bash
dcos marathon group list [OPTION]
```

# 选项

| 名称，简写 | 说明 |
|---------|-------------|
| `--json` | 显示 JSON 格式的数据。|
