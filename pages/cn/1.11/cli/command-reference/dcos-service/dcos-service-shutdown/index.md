# 说明
`dcos service shutdown` 命令让您关闭服务。

# 使用

```bash
dcos service shutdown <service-id> [OPTION]
```

# 选项

无。

# 位置自变量

| 名称，简写 | 说明 |
|---------|-------------|
| `<service-id>` | DC/OS 服务 ID。|

# 父命令

| 命令 | 说明 |
|---------|-------------|
| [dcos service](/cn/1.11/cli/command-reference/dcos-service/) | 管理 DC/OS 服务。|
