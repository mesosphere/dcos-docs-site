# 说明
`dcos job add` 命令让您通过创建 JSON 格式配置文件来添加作业。

# 使用

```bash
dcos job add <job-file> [OPTION]
```

# 位置自变量

| 名称，简写 | 说明 |
|---------|-------------|
| `<job-file>` | 指定以 JSON 为格式的作业的定义。|

# 父命令

| 命令 | 说明 |
|---------|-------------|
|  [dcos job](/cn/1.11/cli/command-reference/dcos-job/)  | 在 DC/OS 中部署和管理作业。|
