# 说明
`dcos job update` 命令让您更新作业。

# 使用

```bash
dcos job update <job-file> [OPTION]
```

# 选项

无。

# 位置自变量

| 名称，简写 | 说明 |
|---------|-------------|
| `<job-file>` | 以 JSON 为格式的作业定义。|
