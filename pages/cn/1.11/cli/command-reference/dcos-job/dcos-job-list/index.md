# 说明
`dcos jobs list` 命令允许您显示所有作业定义。

# 使用

```bash
dcos job list [OPTION]
```

# 选项

| 名称，简写 | 说明 |
|---------|-------------|
| | `--json` | 显示 JSON 格式化列表。 |
