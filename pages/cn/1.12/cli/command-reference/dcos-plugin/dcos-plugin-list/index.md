# 说明

`dcos plugin list` 命令将打印已安装的 CLI 插件列表。

# 使用

```bash
dcos plugin list [flags]
```

# 选项

| 名称 | 说明 |
|---------|------------|
|  `--help, h` | 打印使用。|
| `--json` | JSON 格式的列表。|
