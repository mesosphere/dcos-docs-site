# 说明
`dcos license audit` 命令可以让您检索您的 DC/OS 许可证的审核数据。

# 使用

```bash
dcos license audit [OPTIONS] COMMAND [ARGS]...
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `--help` | 显示此消息后退出。 |
