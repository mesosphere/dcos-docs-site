# Description
The `dcos job update` command allows you to update a job.

# Usage

```bash
dcos job update <job-file> [OPTION]
```

# Options

None.

# Positional arguments

| Name, shorthand |  Description |
|---------|-------------|
| `<job-file>`   |   A JSON-formatted job definition. |

# Parent command

| Command | Description |
|---------|-------------|
| [dcos job](/1.13/cli/command-reference/dcos-job/) |  Deploy and manage jobs in DC/OS. |
