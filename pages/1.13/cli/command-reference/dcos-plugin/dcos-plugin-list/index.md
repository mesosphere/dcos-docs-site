# Description

Print a list of the installed DC/OS plugins.

# Usage

```bash
dcos plugin list [OPTION]
```

# Options

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `--json`   |             |  JSON-formatted list. |
