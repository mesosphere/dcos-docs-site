# Description

Uninstall a plugin.

# Usage

```bash
dcos plugin remove <plugin>
```

# Positional arguments

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `<plugin>`   |             |  Name of the DC/OS plugin. |

# Parent command

| Command | Description |
|---------|-------------|
| [dcos plugin](/1.13/cli/command-reference/dcos-plugin/)   | Install and manage DC/OS CLI plugins. |
