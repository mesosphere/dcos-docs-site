# Description
The `dcos auth` command manages DC/OS identity and access.

# Usage

```bash
dcos auth
```

# Options

| Name, shorthand | Description |
|---------|-------------|-------------|
| `--help, h`   | Display usage. |
| `--info`   |  Display a short description of this subcommand. |
| `--version, v`   | Display version information. |
