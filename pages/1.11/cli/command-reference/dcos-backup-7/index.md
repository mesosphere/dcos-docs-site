## dcos backup
Create backups and restore from them.

```
dcos backup
Usage:
    dcos backup --help
    dcos backup --info
    dcos backup --version
    dcos backup create --label=<label>
    dcos backup restore <id>
    dcos backup list [--json] [<prefix>]
    dcos backup show [--json] <id>
    dcos backup delete <id>
```
