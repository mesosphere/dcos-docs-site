# Description
Get the cluster license status.

# Usage

```bash
dcos license status  [--terms] [--breaches]
```

# Options

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `--terms`   |             |  Displays the terms of contract. |
| `--breaches`   |             |  Displays the number of breaches. |
