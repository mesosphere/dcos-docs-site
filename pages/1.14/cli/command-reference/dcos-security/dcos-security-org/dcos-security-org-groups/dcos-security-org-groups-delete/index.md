# Description

The `dcos security org groups delete` command allows you to remove a group.

# Usage

```
dcos security org groups delete [OPTIONS] GID
```

# Options

| Name |  Description |
|---------|-------------|
|  `-h`, `--help` |  Show this message and exit.|

## Positional Arguments

| Name |  Description |
|---------|-------------|
| `GID` | Group ID. (Required)|
