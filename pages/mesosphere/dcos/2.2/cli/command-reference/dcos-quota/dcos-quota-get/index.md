# Description

The `dcos quota get` command allows you to get an existing quota.

# Usage

```bash
dcos quota get <group> [flags]
```

# Options

| Name |  Description |
|---------|-------------|
| `--help, h`     | Print usage. |
| `--json`   |   JSON-formatted result. |
