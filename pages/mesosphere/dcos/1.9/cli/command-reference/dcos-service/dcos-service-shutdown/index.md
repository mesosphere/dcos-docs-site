# Description
Shutdown a service.

# Usage

```bash
dcos service shutdown <service-id> [OPTION]
```

# Options

None. 

# Positional arguments

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `<service-id>`   |             | The DC/OS service ID. |

# Parent command

| Command | Description |
|---------|-------------|
| [dcos service](/mesosphere/dcos/1.9/cli/command-reference/dcos-service/)   | Manage DC/OS services. |
