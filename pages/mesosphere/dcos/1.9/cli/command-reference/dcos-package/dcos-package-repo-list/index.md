# Description
Print the package repository sources. Possible sources include a local file, HTTPS, and Git.

# Usage

```bash
dcos package repo list [OPTION]
```

# Options

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `--json`   |             |  JSON-formatted data. |
