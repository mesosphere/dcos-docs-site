# Description
Add a group.

# Usage

```bash
dcos marathon group add <group-resource> [OPTION]
```

# Options

None.

# Positional arguments

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `<group-resource>`   |             |  Path to a file or HTTP(S) URL that contains the group's JSON definition. If omitted, the definition is read from stdin. For a detailed description see the [documentation](/mesosphere/dcos/1.9/deploying-services/marathon-api/). |

# Parent command

| Command | Description |
|---------|-------------|
| [dcos marathon](/mesosphere/dcos/1.9/cli/command-reference/dcos-marathon/) | Deploy and manage applications to DC/OS. |
