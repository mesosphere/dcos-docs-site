# Description

The `dcos monitoring prometheus status flags` returns the flag values with which Prometheus was configured.

# Usage

```bash
dcos monitoring prometheus status flags [flags]
```

# Options

| Name |  Description |
|---------|-------------|
| `--help, h`   |   Display usage. |
