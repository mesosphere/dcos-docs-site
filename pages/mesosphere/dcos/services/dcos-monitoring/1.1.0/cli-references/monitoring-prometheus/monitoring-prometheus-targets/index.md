# Description

The `dcos monitoring prometheus targets` command returns an overview of the current state of the Prometheus target discovery.

# Usage

```bash
dcos monitoring prometheus targets [flags]
```

# Options

| Name |  Description |
|---------|-------------|
| `--help, h`   |   Display usage. |
