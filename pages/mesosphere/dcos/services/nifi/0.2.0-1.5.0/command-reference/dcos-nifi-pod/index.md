# Description
The `dcos nifi pod` command displays pod related informations and allows pod operations of the NiFi service.

# Usage

```bash
dcos nifi pod [command]
```

# Options

| Name |  Description |
|---------|-------------|
| `-h, --help`   |  Show context-sensitive help. |
| `-v, --verbose`   |  Enable extra logging of requests/responses |
| `--name="nifi"`   |  Name of the service instance to query |
