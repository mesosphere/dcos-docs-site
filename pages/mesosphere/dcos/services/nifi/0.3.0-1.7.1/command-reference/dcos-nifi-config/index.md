# Description
The `dcos nifi config` command displays configuration related informations of the NiFi service.

# Usage

```bash
dcos nifi config [command]
```

# Options

| Name |  Description |
|---------|-------------|
| `-h, --help`   |  Show context-sensitive help. |
| `-v, --verbose`   |  Enable extra logging of requests/responses |
| `--name="nifi"`   |  Name of the service instance to query |
