# Description
The `dcos nifi update` command displays update related informations and allows update operations of the NiFi service.

# Usage

```bash
dcos nifi update [command]
```

# Options

| Name |  Description |
|---------|-------------|
| `-h, --help`   |  Show context-sensitive help. |
| `-v, --verbose`   |  Enable extra logging of requests/responses |
| `--name="nifi"`   |  Name of the service instance to query |
