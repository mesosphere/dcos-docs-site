# Description
The `dcos nifi state framework_id` command displays the Mesos framework ID.


# Usage

```bash
dcos nifi state framework_id [flags]
```

# Options

| Name |  Description |
|---------|-------------|
| `-h, --help`   |  Show context-sensitive help. |
| `-v, --verbose`   |  Enable extra logging of requests/responses |
| `--name="nifi"`   |  Name of the service instance to query |
