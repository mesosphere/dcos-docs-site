# Description

The `dcos config keys` returns the list of configuration property keys that you can set in a DC/OS configuration file.

# Usage

```bash
dcos config keys [options]
```
# Options

| Name |  Description |
|---------|-------------|
| `--help, h`   |   Display usage. |
| `--quiet, q`   |   Only print configuration keys. |
