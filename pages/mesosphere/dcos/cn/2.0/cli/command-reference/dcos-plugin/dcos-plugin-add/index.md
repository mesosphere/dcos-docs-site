# 说明

`dcos plugin add` 命令允许您添加 CLI 插件。

# 使用

```bash
dcos plugin add <resource> [flags]
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `--update`, `-u` | 插件版本。 |
| | `--help, h` | 打印使用。|

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `<resource>` | 插件资源路径。这可以是 `.zip` 文件或 URL |
