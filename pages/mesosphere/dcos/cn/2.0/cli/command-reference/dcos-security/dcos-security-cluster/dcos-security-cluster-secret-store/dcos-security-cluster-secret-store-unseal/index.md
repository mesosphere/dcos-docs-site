# 说明

`dcos security cluster secret-store unseal` 命令允许您拆封给定存储库。

# 使用

```
dcos security cluster secret-store unseal [OPTIONS] STORE_ID KEY
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `-j`, `--json` | JSON 格式的输出数据。 |
| `-h`, `--help`| 显示此消息并退出。|

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `STORE_ID` | 密钥存储库的 ID。 |
