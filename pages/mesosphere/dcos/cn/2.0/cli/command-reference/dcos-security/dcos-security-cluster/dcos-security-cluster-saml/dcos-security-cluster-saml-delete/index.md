# 说明

`dcos security cluster saml delete` 命令允许您删除 SAML 提供程序配置。

# 使用

```
dcos security cluster saml delete [OPTIONS] SAML_ID
```


# 选项

| 名称 | 说明 |
|-------------------|------------------|
| `-h`, `--help`| 显示此消息并退出。|


## 位置自变量

| 名称 | 说明 |
|--------|------------------|
| `SAML_ID` | SAML 提供程序的 ID。 |
