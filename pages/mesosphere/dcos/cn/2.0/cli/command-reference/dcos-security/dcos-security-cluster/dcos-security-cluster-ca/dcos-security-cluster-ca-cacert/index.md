# 说明

`dcos security cluster ca cacert` 命令将获取 PEM 编码的签名 CA 证书（根 CA 证书或中间 CA 证书）。

# 使用

```
dcos security cluster ca cacert [OPTIONS]
```


# 选项

| 名称 | 说明 |
|---------|-------------|
| `-h`, `--help`| 显示此消息并退出。|
