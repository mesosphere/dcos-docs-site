# 说明

`dcos security cluster directory import_user` 命令从配置的目录 (LDAP) 后端导入用户。


# 使用

```
dcos security cluster directory import_user [OPTIONS] UID
```


# 选项

| 名称 | 说明 |
|----------|---------|
| `-h`, `--help`| 显示此消息并退出。|

## 位置自变量

| 名称 | 说明 |
|--------|-------------------|
| `UID` | 用户 ID。 |
