# 说明

`dcos security org groups add_user` 命令允许您将用户添加到指定用户组。

# 使用

```
dcos security org groups add_user [OPTIONS] GID UID
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `-h`, `--help`| 显示此消息并退出。|


## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `GID` | 组 ID。（必填）|
| `UID` | 用户 ID。（必填）|
