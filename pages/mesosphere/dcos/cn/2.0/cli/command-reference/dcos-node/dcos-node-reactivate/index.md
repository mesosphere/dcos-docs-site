# 说明

`dcos node reactivate` 命令允许您重新激活 Mesos 代理节点。

# 使用

```bash
dcos node reactivate <mesos-id>
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `--help, h` | 显示用法。 |

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `<mesos-id>` | 节点的代理 ID。|
