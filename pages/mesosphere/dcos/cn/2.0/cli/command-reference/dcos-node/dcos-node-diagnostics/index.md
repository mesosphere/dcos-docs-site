# 说明
`dcos node diagnostics` 命令允许您查看诊断捆绑包的详细信息。

**自 DC/OS 2.0 以来，此命令已弃用，请使用 `dcos diagnostics` 代替。**

# 使用

```bash
dcos node diagnostics (--list | --status | --cancel) [--json]
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `--cancel` | 取消正在运行的诊断工作。|
| `--list` | 列出可用的诊断捆绑包。|
| `--json` | 显示 JSON 格式的数据。|
| `--status` | 显示诊断工作状态。|
| `--help, h` | 显示用法。 |

dcos node diagnostics --list
