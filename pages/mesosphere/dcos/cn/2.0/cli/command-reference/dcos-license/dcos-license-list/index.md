# 说明
`dcos license list` 命令将列出与群集关联的所有许可证。此命令默认输出为 `stdout`。

# 使用

```bash
dcos license list [OPTIONS]
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `--help` | 显示此消息并退出。|
