# 说明
`dcos cluster rename` 命令将重命名配置的群集。

# 使用

```bash
dcos cluster rename <cluster> <name> [flags]
```

# 选项
| 名称 | 说明 |
|---------|-------------|
| `-h`，`--help` | 此命令的帮助。 |

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `<cluster>` | 已连接的群集的名称 |
| `<name>` | 连接的群集的新名称 |

有关示例，请参阅 [群集连接](/mesosphere/dcos/2.0/administering-clusters/multiple-clusters/cluster-connections/)。
