您可以使用自动扩展来根据使用情况自动增加或减少计算资源，从而只使用所需的资源。以下是一些教程，向您展示如何为您的服务实施自动扩展。

<p class="message--important"><strong>重要信息：</strong>自动扩展 <strong>仅</strong> 在“禁用”安全模式后，适用于 DC/OS Open Source 或 DC/OS Enterprise。</p>
