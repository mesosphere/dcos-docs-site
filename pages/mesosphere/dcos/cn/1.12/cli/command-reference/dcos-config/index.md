# 说明

`dcos config` 命令管理您在运行 [dcos cluster setup](/mesosphere/dcos/1.12/cli/command-reference/dcos-cluster/dcos-cluster-setup/) 时创建的 DC/OS 配置文件。该配置文件位于 `~/.dcos/clusters/<cluster_id>/dcos.toml`。

# 使用

```bash
  dcos config [command]
```

# 选项

| 名称 | 说明 |
|---------|-------------|
|  `--help, h` | 显示使用情况。|
