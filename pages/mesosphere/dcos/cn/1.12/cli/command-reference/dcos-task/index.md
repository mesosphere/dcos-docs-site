# 说明

`dcos task` 命令可让您管理 DC/OS 任务。

# 使用

```bash
dcos task [OPTIONS]
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `-h`, `--help`  | 打印使用情况。|
| `--info` | 打印该子命令的简短描述。|
| `--version` | 打印版本信息。|
