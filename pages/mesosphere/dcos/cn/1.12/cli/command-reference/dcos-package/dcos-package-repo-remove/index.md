# 说明
`dcos package repo remove` 命令让您可以从 DC/OS 中删除软件包存储库。

# 使用

```bash
dcos package repo remove <repo-names>...
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `-h`, `--help` | 显示用法。|

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `<repo-name>` | 软件包存储库的名称。例如， `Universe`。|
