# 说明

`dcos package repo import` 命令让您可以导入包含带有 `<repo-file>` 标识的软件包存储库的文件。

# 使用

```
dcos package repo import <repos-file>
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `-h`, `--help` | 显示用法。|

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `<repo-file>` | 包含软件包存储库的文件，列出的格式为 `dcos package list --json`。|
