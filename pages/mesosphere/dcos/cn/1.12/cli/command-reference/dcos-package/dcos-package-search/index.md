# 说明
`dcos package search` 命令让您可以搜索软件包存储库。

# 使用

```bash
dcos package search [<query> --json]
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `-h`, `--help` | 显示用法。|
| `--json` | 将输出显示为 JSON 格式的数据。 |

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `<query>` | 用于搜索软件包存储库的模式。您可以使用完整值或部分值。|
