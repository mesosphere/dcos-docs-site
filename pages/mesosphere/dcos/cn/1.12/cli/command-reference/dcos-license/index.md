`dcos license` 命令可以让您查看许可证的状态、审核您的许可证以及获取或更新许可证。

# 使用

```
dcos license [OPTIONS] COMMAND [ARGS]...
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `--version` | 显示版本并退出。|
| `--info` | 显示有关子命令的信息。|
| `--help` | 显示此消息并退出。|
