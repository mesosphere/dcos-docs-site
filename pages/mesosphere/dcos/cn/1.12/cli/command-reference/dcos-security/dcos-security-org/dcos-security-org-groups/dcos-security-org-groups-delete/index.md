# 说明

`dcos security org groups delete` 命令让您删除组。

# 使用

```
dcos security org groups delete [OPTIONS] GID
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `-h`, `--help` | 显示此消息并退出。|

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `GID` | 组 ID。（必填）|
