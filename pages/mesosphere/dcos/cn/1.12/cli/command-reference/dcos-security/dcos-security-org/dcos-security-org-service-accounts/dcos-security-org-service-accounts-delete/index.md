# 说明

`dcos security org service-accounts delete` 命令让您删除由服务帐户 ID (SID) 标识的服务帐户。

# 使用

```
dcos security org service-accounts delete [OPTIONS] SID
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `-h`, `--help` | 显示此消息并退出。|

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `SID` | 服务账户 ID。（必填）|
