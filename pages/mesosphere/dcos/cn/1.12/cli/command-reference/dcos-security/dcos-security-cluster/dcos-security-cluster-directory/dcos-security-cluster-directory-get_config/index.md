# 说明

`dcos security cluster directory get_config` 命令将检索当前 LDAP 配置。

# 选项

| 名称 | 说明 |
|----------|---------------|
|`-j`, `--json`| JSON 格式的输出数据|
|`-h`, `--help`| 显示此消息并退出|

# 使用

```
dcos security cluster directory get_config [OPTIONS]
```
