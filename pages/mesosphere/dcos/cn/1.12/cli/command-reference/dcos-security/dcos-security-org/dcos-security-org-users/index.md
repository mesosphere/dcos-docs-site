# 说明

`dcos security org users` 命令可让您管理用户权限。

# 使用

```bash
dcos security org users [OPTIONS] COMMAND [ARGS]...
```


# 选项
 
| 名称 | 说明 |
|---------|-------------|
| `-h`, `--help` | 显示此消息并退出。|
| `UID` | 用户 ID。（必填）|
