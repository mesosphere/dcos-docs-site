# 说明

`dcos security org service-accounts` 命令让您可以管理安全帐户。

# 使用

```
dcos security org service-accounts [OPTIONS] COMMAND [ARGS]...
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `-h`, `--help` | 显示此消息并退出。|

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `SID` | 服务账户 ID。（必填）|
