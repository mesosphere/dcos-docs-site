# 说明

Mesosphere DC/OS 是一个围绕 Apache Mesos 构建的分布式操作系统。这种实用工具提供用于轻松管理 DC/OS 安装的工具。

`dcos` 命令帮助您管理 DC/OS 安装。

# 使用

``` bash
dcos [command]
```

# 选项

| 名称 | 说明 |
|---------|-------------|-------------|
| `--help, h` | 显示使用帮助。 |
| `--version` | 打印版本信息。|
| `-v`，`-vv` | 输出详细信息（详细或非常详细）。|
