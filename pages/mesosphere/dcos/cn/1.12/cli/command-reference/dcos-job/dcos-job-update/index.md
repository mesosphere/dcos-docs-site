# 说明
`dcos job update` 命令可以让您更新作业。

# 使用

```bash
dcos job update <job-file>
```

# 选项

| 名称 | 说明 |
|---------|-------------|
|`-h`，`--help` | 打印用法。 |

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `<job-file>` | 作业的 JSON 格式文件。 |
