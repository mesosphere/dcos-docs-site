# 说明
`dcos marathon pod kill` 命令让您终止一个或多个正在运行的 pod 实例。

# 使用

```bash
dcos marathon pod kill <instance-ids> <pod-id> [OPTION]
```

# 选项

无。

# 位置自变量

| 名称，简写 | 说明 |
|---------|-------------|
| `<instance-ids>` | 一个或多个 pod 实例 ID 的列表，以空格分隔。|
| `<pod-id>`   |  pod ID。 您可以使用以下命令查看pod ID列表 `dcos marathon pod list` 命令。|
