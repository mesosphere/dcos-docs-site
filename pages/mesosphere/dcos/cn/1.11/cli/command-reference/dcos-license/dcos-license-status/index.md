# 说明
`dcos license status` 命令将显示集群许可证状态。

# 使用

```bash
dcos license status  [--terms] [--breaches]
```

# 选项

| 名称，简写 | 说明 |
|---------|-------------|
| | `--terms` | 显示合同条款。|
| | `--breaches` | 显示违规次数。|

# 父命令

| 命令 | 说明 |
|---------|-------------|
| [dcos license](/mesosphere/dcos/cn/1.11/cli/command-reference/dcos-license/) | 管理 DC/OS 群集许可证。 |
