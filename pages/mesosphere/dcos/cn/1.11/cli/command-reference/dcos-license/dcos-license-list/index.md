# 说明
`dcos license list` 命令将显示所有集群许可证。

# 使用

```bash
dcos license list --output <file_path>
```

# 选项

| 名称，简写 | 说明 |
|---------|-------------|
| | `--output` | 将许可证存储在文件中。|


# 位置自变量

| 名称，简写 | 说明 |
|---------|-------------|
| `<file_path>` | 存储许可证的文件的路径。|


# 父命令

| 命令 | 说明 |
|---------|-------------|
| [dcos license](/mesosphere/dcos/cn/1.11/cli/command-reference/dcos-license/) | 管理 DC/OS 群集许可证。 |
