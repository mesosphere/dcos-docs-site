# 说明
`dcos cluster` 命令允许您管理与 DC/OS 集群的连接。

# 使用

```bash
dcos cluster
```

# 选项

| 名称，简写 | 说明 |
|---------|-------------|
| | `--help, h` | 显示使用情况。|
| | `--info` | 显示此子命令的简短说明。|
| | `--version, v` | 显示版本信息。 |
