# 说明
`dcos cluster rename` 命令让您重命名集群。

# 使用

```bash
dcos cluster rename <name> <new-name>
```

# 位置自变量

| 名称，简写 | 说明 |
|---------|-------------|
| `<name>` | 已连接的集群的名称。 |
| `<new-name>` | 已连接的集群的新名称。|
