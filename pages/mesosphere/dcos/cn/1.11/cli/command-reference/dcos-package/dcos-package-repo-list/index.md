# 说明
`dcos package repo list` 命令显示软件包存储库源。可能的源包括本地文件、HTTPS 和 Git。

# 使用

```bash
dcos package repo list [OPTION]
```

# 选项

| 名称，简写 | 说明 |
|---------|-------------|
| `--json` | JSON 格式的数据。|
