# 说明
`dcos package` 命令让您安装和管理 DC/OS 软件包。

# 使用

```bash
dcos package
```

# 选项

| 名称，简写 | 说明 |
|---------|-------------|
| `--config-schema` | 显示子命令的配置架构。|
| `--help, h` | 显示用法。 |
| `--info` | 显示该子命令的简短描述。|
| `--version, v` | 显示版本信息。|
