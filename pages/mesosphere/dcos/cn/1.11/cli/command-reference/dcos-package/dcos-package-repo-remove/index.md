# 说明
`dcos package repo remove` 命令让您从 DC/OS 中删除软件包存储库。

# 使用

```bash
dcos package repo remove <repo-name> [OPTION]
```

# 选项

| 名称，简写 | 说明 |
|---------|-------------|
| `<repo-name>`   |   包存储库的名称。 例如, `Universe`. |
