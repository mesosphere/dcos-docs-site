# 说明
`dcos job` 命令允许您在 DC/OS 中部署和管理作业。

# 使用

```bash
dcos job
```

# 选项

| 名称，简写 | 说明 |
|---------|--------------|
| | `--help, h` | 显示使用情况。|
| | `--info` | 显示此子命令的简短说明。|
| | `--version, v` | 显示版本信息。 |
