# 说明
`dcos job schedule update` 命令让您更新作业时间表。

# 使用

```bash
dcos job schedule update <job-id> <schedule-file> [OPTION]
```

# 选项

无。

# 位置自变量

| 名称，简写 | 说明 |
|---------|-------------|
| `<job-id>`   |  指定作业ID。 您可以使用以下命令查看作业ID `dcos job list` 命令。|
| `<schedule-file>` | 以 JSON 为格式的作业时间表文件。|
