# 说明
`dcos node` 命令让您查看 DC/OS 节点信息。

# 使用

```bash
dcos node [OPTION]
```

# 选项

| 名称，简写 | 说明 |
|---------|-------------|
| `--help, h` | 显示用法。 |
| `--info` | 显示该子命令的简短描述。|
| `--json` | 显示 JSON 格式的数据。|
| `--version, v` | 显示版本信息。|
