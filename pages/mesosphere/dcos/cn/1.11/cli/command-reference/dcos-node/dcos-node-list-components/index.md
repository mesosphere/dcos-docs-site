# 说明
`dcos node list-components` 命令在指定节点上显示可用的 DC/OS 组件列表。

# 使用

```bash
dcos node list-components [OPTION]
```

# 选项

| 名称，简写 | 说明 |
|---------|-------------|
| `--json` | 显示 JSON 格式的数据。|
| `--leader` | 主节点。|
| `--mesos-id=<mesos-id>` | 节点的代理 ID。|

|---------|-------------|
