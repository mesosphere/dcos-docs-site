DC/OS Web 界面提供了 DC/OS 集群的丰富图形视图。通过 Web 界面，可以查看整个集群和 DC/OS 服务的当前状态。Web 界面作为 DC/OS 安装的一部分安装。

![Dashboard](/mesosphere/dcos/cn/1.11/img/dashboard-ee.png)

图 1. 仪表板

# Web 界面导航菜单
在左侧，您将看到 Web 界面导航菜单：
- [仪表板](/mesosphere/dcos/cn/1.11/gui/dashboard/) （默认视图）
- [服务](/mesosphere/dcos/cn/1.11/gui/services/)
- [作业](/mesosphere/dcos/cn/1.11/gui/jobs/)
- [目录](/mesosphere/dcos/cn/1.11/gui/catalog/)
- [节点](/mesosphere/dcos/cn/1.11/gui/nodes/)
- [网络](/mesosphere/dcos/cn/1.11/gui/networking/)
- [密钥](/mesosphere/dcos/cn/1.11/gui/secrets/)（仅适用于 DC/OS Enterprise ）[enterprise type="inline" size="small" /]
- [集群](/mesosphere/dcos/cn/1.11/gui/cluster/)
- [组件](/mesosphere/dcos/cn/1.11/gui/components/)
- [设置](/mesosphere/dcos/cn/1.11/gui/settings/)
- [组织](/mesosphere/dcos/cn/1.11/gui/organization/)
