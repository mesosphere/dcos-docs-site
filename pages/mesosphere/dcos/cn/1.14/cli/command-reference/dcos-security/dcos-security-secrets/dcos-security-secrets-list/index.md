# 说明

`dcos security secrets list` 命令将列出给定路径中存储的所有密钥。

# 使用

```
dcos security secrets list [OPTIONS] PATH
```

# 选项

| 名称 | 说明 |
|------------------|----------------------|
|`-s`, `--store-id <text>` | 要使用的密钥后端。|
|`-j`, `--json` | JSON 格式的输出数据。|
| `-h`, `--help` | 显示此消息并退出。 |

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `PATH` | 密钥路径的 URL 或 IP 地址。 |
