# 说明

`dcos security org groups show` 命令将显示一个或多个组的基本信息。

# 使用

```
dcos security org groups show [OPTIONS] [GIDS]...
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `-h`, `--help`| 显示此消息并退出。|
| `-j`, `--json` | JSON 格式的输出数据。 |

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `GID` | 组 ID。（必填）|
