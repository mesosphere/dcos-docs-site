# 说明
`dcos cluster attach` 命令将把 CLI 附加到已连接或 [链接](/mesosphere/dcos/1.14/cli/command-reference/dcos-cluster/dcos-cluster-link/) 的群集。当您运行 [`dcos cluster setup`](/mesosphere/dcos/1.14/cli/command-reference/dcos-cluster/dcos-cluster-setup/) 命令时，群集自动被附加。

# 使用

```bash
dcos cluster attach <cluster> [flags]
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| | `--help, h` | 显示使用情况。|

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `<cluster>` | 群集的 ID（必填）|
