# 说明
`dcos marathon task list` 命令显示所有任务的列表。

# 使用

```bash
dcos marathon task list [--json|--quiet] [<app-id>]
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `--json` | 显示 JSON 格式的数据。|
| `-q`，`--quiet` | 仅显示列表的 ID。 |
| `-h`，`--help` | 显示有关此命令用法的信息。 |


## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `<app-id>` | 应用程序 ID。|

| 命令 | 说明 |
|---------|-------------|
