# 说明
`dcos backup list` 命令将列出所有已知备份。

# 使用

```bash
dcos backup list [prefix] [flags]
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `--help, h` | 显示此命令的帮助。 |
| `--json` | 以 JSON 格式显示输出。 |

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `prefix` | 识别附加到文件名的前缀，用于排序。 |
