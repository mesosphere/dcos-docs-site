# 说明

`dcos security cluster secret-store seal-status` 命令显示密钥存储库的密封状态。

# 使用

```
dcos security cluster secret-store seal-status [OPTIONS] STORE_ID
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `-j`, `--json` | JSON 格式的输出数据。 |
| `-h`, `--help`| 显示此消息并退出。|


## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `STORE_ID` | 密钥存储库 ID。（必填）|
