# 说明

`dcos security org groups members` 命令将列出组中的所有成员。

# 使用

```
dcos security org groups members [OPTIONS] GID
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `-h`, `--help`| 显示此消息并退出。|
| `-j`, `--json` | JSON 格式的输出数据。 |

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `GID` | 组 ID。（必填）|
