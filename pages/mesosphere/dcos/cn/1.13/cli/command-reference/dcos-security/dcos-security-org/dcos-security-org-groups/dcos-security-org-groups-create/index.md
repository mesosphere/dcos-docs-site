# 说明
`dcos security org groups create` 命令允许您创建新组。

# 使用

```bash
dcos security org groups create [OPTIONS] GID
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `d`, `--description <text>` | 组描述。|
| `-h`, `--help`| 显示此消息并退出。|

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `GID` | 组 ID。（必填）|
