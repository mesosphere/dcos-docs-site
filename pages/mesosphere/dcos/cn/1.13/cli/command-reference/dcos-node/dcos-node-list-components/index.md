# 说明
`dcos node list-components` 命令在指定节点上显示可用的 DC/OS 组件列表。

# 使用

```bash
dcos node list-components [--leader --mesos-id=<mesos-id> --json]
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `--help, h` | 显示用法。 |
| `--json` | 显示 JSON 格式的数据。|
| `--leader` | 主节点。|
| `--mesos-id=<mesos-id>` | 节点的代理 ID。|

| 命令 | 说明 |
|---------|-------------|
