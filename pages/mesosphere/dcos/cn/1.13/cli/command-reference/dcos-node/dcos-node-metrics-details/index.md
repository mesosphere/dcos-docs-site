# 说明

`dcos node metrics details` 命令打印一张表格，显示 <mesos-id> 指定的代理节点的所有度量标准。

# 使用

```
dcos node metrics details <mesos-id>  [--json]
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `--help, h` | 显示用法。 |
| `--json` | 显示 JSON 格式的数据。|
| `--mesos-id=<mesos-id>` | 节点的代理 ID。|
