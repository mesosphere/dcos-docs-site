# 说明

`dcos node decommission` 命令允许您将代理程序标记为已消失。

# 使用

```bash
dcos node decommission <mesos-id>
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `--help, h` | 显示用法。 |
| `--mesos-id=<mesos-id>` | 节点的代理 ID。|
