# 说明
`dcos jobs list` 命令允许您显示所有作业定义。

# 使用

```bash
dcos job list [--json|--quiet]
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `-h`，`--help` | 显示此命令的使用。 |
| `--json` | 打印 JSON 格式列表而不是表格。 |
| `-q`，`--quiet` | 表示仅导致运行 ID 阵列的静默模式。 |
