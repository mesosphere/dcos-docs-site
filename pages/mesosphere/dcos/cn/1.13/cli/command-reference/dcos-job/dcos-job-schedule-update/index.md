# 说明
`dcos job schedule update` 命令允许您更新作业的时间表。

# 使用

```bash
dcos job schedule update <job-id> <schedule-file>
```

# 选项

| 名称 | 说明 |
|---------|-------------|
|`-h`，`--help` | 打印用法。 |

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `<job-id>` | 指定作业 ID。您可以使用 `dcos job list` 命令查看作业 ID。|
| `<schedule-file>` | JSON 格式的作业时间表文件。|
