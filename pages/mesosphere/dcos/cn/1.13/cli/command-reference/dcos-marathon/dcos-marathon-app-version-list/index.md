# 说明

`dcos marathon app version list` 命令允许您列出应用程序的版本历史记录。

# 使用

```bash
dcos marathon app version list [--max-count=<max-count>] <app-id>
```

# 选项

| 名称 | 说明 |
|---------|-------------|
| `-h`，`--help` | 显示有关此命令用法的信息。 |
| `--max-count=<max-count>` | 获取和返回的最大条目数。|

## 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `<app-id>` | 应用程序 ID。|

|---------|-------------|
