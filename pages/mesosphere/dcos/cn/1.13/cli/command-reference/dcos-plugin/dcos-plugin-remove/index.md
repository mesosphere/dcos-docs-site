# 说明

`docs plugin remove` 命令将卸载插件。

# 使用

```bash
dcos plugin remove <plugin>
```

# 位置自变量

| 名称 | 说明 |
|---------|-------------|
| `<plugin>` | DC/OS 插件的名称。 |
