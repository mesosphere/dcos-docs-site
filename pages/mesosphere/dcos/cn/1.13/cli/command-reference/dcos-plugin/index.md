# 说明

`dcos plugin` 命令允许您安装和管理 DC/OS CLI 插件。和 `dcos package` 不同，`dcos plugin` 仅用于在计算机上本地管理 CLI 插件，并且对群集没有影响。

# 使用

```bash
dcos plugin [command]
```

# 选项

| 名称 | 说明 |
|-----------------|-------------|
| | `--help, h` | 打印使用。|
