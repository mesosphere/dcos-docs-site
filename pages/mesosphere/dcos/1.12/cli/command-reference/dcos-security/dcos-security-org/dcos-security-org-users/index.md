# Description

The `dcos security org users` commands let you manage your user permissions.

# Usage

```bash
dcos security org users [OPTIONS] COMMAND [ARGS]...
```


# Options
 
| Name |  Description |
|---------|-------------|
|  `-h`, `--help` |  Show this message and exit.|
| `UID` | User ID. (Required)|
