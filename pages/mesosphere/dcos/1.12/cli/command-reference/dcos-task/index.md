# Description

The `dcos task` commands let you manage DC/OS tasks.

# Usage

```bash
dcos task [OPTIONS]
```

# Options

| Name |  Description |
|---------|-------------|
| `-h`, `--help`  |    Print usage. |
| `--info` |  Print a short description of this subcommand.|
| `--version` |  Print version information.|
