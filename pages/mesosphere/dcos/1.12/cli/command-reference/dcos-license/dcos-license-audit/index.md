# Description
The `dcos license audit` command allows you to retrieve audit data for your DC/OS licenses.

# Usage

```bash
dcos license audit [OPTIONS] COMMAND [ARGS]...
```

# Options

| Name |  Description |
|---------|-------------|
| `--help`   |  Show this message and exit. |
