# Description
Kill the job.

# Usage

```bash
dcos job kill <job-id> [OPTION]
```

# Options

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `run-id`   |             |  The job run ID. |
| `--all`   |             |  Kill all job runs. |

# Positional arguments

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `<job-id>`   |             |  Specify the job ID. |
