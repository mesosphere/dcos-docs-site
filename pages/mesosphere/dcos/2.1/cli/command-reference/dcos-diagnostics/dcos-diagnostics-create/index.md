# Description
The `dcos diagnostics create` command allows you to create a diagnostics bundle.

# Usage

```bash
dcos diagnostics create (<nodes>)
```

# Options

| Name |  Description |
|---------|-------------|
| `--help, h`   |   Displays usage. |

# Example

```bash
$ dcos diagnostics create
a697769a-2d5d-4b2a-a2ec-055b9fe9eecf
```
