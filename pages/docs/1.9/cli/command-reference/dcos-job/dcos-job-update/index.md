# Description
Update a job.

# Usage

```bash
dcos job update <job-file> [OPTION]
```

# Options

None.

# Positional arguments

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `<job-file>`   |             |  A JSON-formatted job definition. |

| Command | Description |
|---------|-------------|
